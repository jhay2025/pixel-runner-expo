# 🎮 Pixel Runner

A retro-style platformer game built with **Expo** & **React Native**.

## 🤖 About

Pixel Runner is a mobile platformer game where you control a cute robot through 3 challenging levels filled with enemies, coins, and obstacles.

## 🕹️ Controls

- **◀ ▶** Move left/right
- **▲** Jump
- Jump on enemies to defeat them!

## 🌟 Features

- 3 unique levels: Forest Ruins, Crystal Caves, Sky Fortress
- Coins to collect
- Enemies with patrol AI
- Lava obstacles
- Lives system
- Score tracking
- Touch controls optimized for mobile

## 🚀 How to Run

### Option 1: Expo Snack (No install needed)
1. Go to [snack.expo.dev](https://snack.expo.dev)
2. Import this project
3. Run on your device with Expo Go

### Option 2: Local Development
```bash
npm install
npx expo start
```

## 📱 Tech Stack

- Expo SDK 52
- React Native
- react-native-game-engine
- Custom physics engine
